Sharelock

Access to the Sharelock Holmes website. The main utility to use is shock-daily

General Instructions

Fixing the StatsList.csv file

Download StatsList.csv to ~/Downloads. Then rectify problems by typing: csv2epics

This places a clean CSV file in int/misc/clean-statslist.csv

Downloading

To download all the epics in clean-statslist.csv: fetch -c

carding

This is the process whereby downloaded htm files are decoded and turned into "cards": shock-daily -r

Calcs

To perform calculations, see calcs.

Tabulate calcs

Create a single table with all the calculations: python calcs-csv.py

Filtering

The Fortran program works out a series of tests. To see all the results for which a certain test is true, execute a command like 
   cat `grep -l "T001 T" data/calcs/* ` | less
Replace T001 with whatever test interests you.

Recipes

Download all the epics in StatsList.csv: csv2epics | getit -i

Determine percentiles for a given column: 

    csvtool namedcol RS_Month `shname` | sort -n | tac| pnl | less
